## How to install the project
### - Requirements

- Node version: 14.18.1
### - Process

- Clone the project 
```
git clone git@github.com:adavidarevalo/aws_practice_exam.git
```
- Install dependencies
```
npm i 
```
- Start the project
```
npm run dev
```
